# User Provided Header
## S2B2S — Repomix Codebase Snapshot (v0.10)

Speech-to-Brain-to-Speech desktop app (Tauri 2 + Rust + React/TypeScript).
Combines STT (9 engine types, 11 variants), LLM Brain (10 providers), and TTS (8 engines) into
a unified voice assistant. ~15,000 lines Rust, ~10,000 lines TypeScript.

Three pipelines: Dictation (speak→paste), Read Aloud (select→hear),
Conversation (speak→LLM→hear). 20-language i18n, system tray, global shortcuts.

See android-port-plan.md for Android on-device port plans.
See improvement-plan.md for streaming STT/TTS analysis.
See reference_links.md for 70+ project reference curated list.
